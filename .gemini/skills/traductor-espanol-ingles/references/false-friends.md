# Falsos amigos ES ↔ EN

Tabla de referencia para evitar los errores más frecuentes de traducción técnica y profesional. Consultar siempre antes de finalizar una traducción larga.

**Cómo se lee la tabla:** la columna del medio es lo que el término **NO** significa (la traducción incorrecta intuitiva). La columna de la derecha es la traducción correcta según contexto.

---

## 1. EN → ES (palabras inglesas que se traducen mal al español)

| Inglés | NO significa | Sí significa | Ejemplo correcto |
|---|---|---|---|
| **actually** | actualmente | de hecho, en realidad | "Actually, the issue is in the network" → "**De hecho**, el problema está en la red" |
| **eventually** | eventualmente (= ocasionalmente) | finalmente, con el tiempo, al final | "The system will eventually recover" → "El sistema se recuperará **al final**" |
| **sensible** | sensible | sensato, razonable | "A sensible approach" → "Un enfoque **sensato**" |
| **library** | librería (= bookstore) | biblioteca (en general) / librería (uso establecido en dev) | "Python library" → "**librería** de Python" (uso aceptado) |
| **support** (verbo) | soportar | dar soporte, apoyar, admitir | "We support TLS 1.3" → "**Admitimos** TLS 1.3" / "Damos soporte a TLS 1.3" — no "soportamos" |
| **support** (sustantivo) | soporte físico | apoyo, soporte técnico | "Customer support" → "Soporte al cliente" ✓ |
| **realize** | realizar | darse cuenta | "We realized the bug was in the cache" → "**Nos dimos cuenta** de que el bug estaba en la caché" |
| **realize** (financiero) | — | realizar (un beneficio, una venta) | OK en contexto financiero |
| **assist** | asistir (a un evento) | ayudar | "Please assist the customer" → "Por favor, **ayuda** al cliente" |
| **attend** | atender | asistir (a un evento) | "Attend the meeting" → "**Asistir** a la reunión" |
| **compromise** (sust.) | compromiso | concesión, componenda | "We reached a compromise" → "Llegamos a un **acuerdo / compromiso negociado**" |
| **compromise** (verbo en seguridad) | comprometer (figurado) | comprometer la seguridad de, vulnerar | "The system was compromised" → "El sistema **fue comprometido / vulnerado**" |
| **commitment** | compromiso (en sentido vago) | compromiso firme, dedicación | "Strong commitment" → "Compromiso **firme**" |
| **pretend** | pretender | fingir | "Don't pretend you didn't know" → "No **finjas** que no lo sabías" |
| **pretend** (formal) | pretender (= aspirar) | aspirar a, intentar | raro; usar "intend" o "aim" en EN moderno |
| **consistent** | consistente (físicamente denso) | coherente, uniforme | "Consistent results" → "Resultados **coherentes**" |
| **disgust** | disgusto | asco, repugnancia | "He felt disgust" → "Sintió **asco**" |
| **suburb** | suburbio (degradado) | área residencial periférica | "The suburbs of Madrid" → "Las **afueras / la periferia** de Madrid" |
| **introduce** | introducir (= meter dentro) | presentar | "Introduce the new policy" → "**Presentar** la nueva política" |
| **discuss** | discutir (= pelear) | tratar, debatir, analizar | "Discuss the proposal" → "**Tratar / analizar** la propuesta" |
| **argument** | argumento (de novela) | argumento (en código) ✓ / discusión / razonamiento | en programación está bien `argumento` |
| **argument** (general) | — | discusión, razonamiento | "We had an argument" → "Tuvimos una **discusión**" |
| **policy** | política (en sentido partidista) | política, normativa | "Security policy" → "Política / normativa de seguridad" ✓ |
| **politics** | políticas | la política (actividad) | "Office politics" → "Las **politiqueos / juegos de poder** de la oficina" |
| **deception** | decepción | engaño | "Phishing deception" → "**Engaño** de phishing" |
| **disappointment** | desilusión leve | decepción | "It was a disappointment" → "Fue una **decepción**" |
| **success** | suceso | éxito | "The migration was a success" → "La migración fue un **éxito**" |
| **event** | evento (siempre) | evento / suceso / hecho | en logs: `events` = sucesos/eventos; en contexto BBDD `wait events` = eventos de espera ✓ |
| **exit** | éxito | salida | "Exit code" → "**Código de salida**" |
| **rope** | ropa | cuerda, soga | — |
| **carpet** | carpeta | alfombra | — |
| **fabric** | fábrica | tela / tejido / `fabric` (en redes/cómputo) | "Storage fabric" → "**Tejido** de almacenamiento / `fabric`" |
| **embarrassed** | embarazada | avergonzado | crítico |
| **molest** | molestar | abusar (sexualmente) | crítico — nunca usar como "molestar" |
| **constipated** | constipado | estreñido | crítico en contexto médico |
| **abstract** | abstracto (siempre) | abstracto / resumen (de un paper) | "Read the abstract" → "Lee el **resumen**" |
| **conference** | conferencia (charla) | conferencia / congreso | "Tech conference" → "**Congreso** tecnológico" (mejor que "conferencia") |
| **conductor** | conductor (de coche) | director (de orquesta) / revisor (tren) | en eléctrica sí = conductor ✓ |
| **fastidious** | fastidioso | meticuloso, exigente | "Fastidious documentation" → "Documentación **meticulosa**" |
| **idiom** | idioma | modismo | "An English idiom" → "Un **modismo** inglés" |
| **language** | — | idioma / lenguaje | en programación: `language` = lenguaje ✓; en lingüística: `language` = idioma |
| **notice** (sust.) | noticia | aviso, notificación | "Legal notice" → "**Aviso** legal" |
| **news** | nuevas | noticias | "Breaking news" → "Última hora / noticias de última hora" |
| **lecture** | lectura | charla, conferencia, clase magistral | "Attend a lecture" → "Asistir a una **clase magistral**" |
| **reading** | — | lectura ✓ | OK |
| **demand** | demandar (jurídico) | exigir, requerir | "High demand" → "Alta **demanda**" ✓; "We demand quality" → "**Exigimos** calidad" |
| **sue** | suelo / Sue (nombre) | demandar (jurídico) | — |
| **sympathetic** | simpático | comprensivo, empático | "Be sympathetic" → "Sé **empático / comprensivo**" |
| **estate** | estado | finca / patrimonio | "Real estate" → "Inmuebles / bienes raíces" |
| **eventually** | eventualmente (= ocasionalmente) | al final, finalmente, con el tiempo | (repetido por su importancia) |
| **resume** (verbo) | resumir | reanudar | "Resume the deployment" → "**Reanudar** el despliegue" |
| **resumé / résumé** | resumen | currículum (CV) | en EN americano `resume` = CV |
| **summary** | sumario (jurídico) | resumen | "Executive summary" → "**Resumen** ejecutivo" |
| **terrific** | terrorífico | estupendo, fantástico | "Terrific job!" → "**¡Trabajo excelente!**" |
| **terrible** | — | terrible / horrible ✓ | OK |
| **petrol** | petróleo | gasolina | (UK) |
| **oil** | aceite (siempre) | aceite / petróleo | contexto |
| **vacancy** | vacaciones | vacante (puesto libre) | "Job vacancy" → "**Vacante** laboral" |
| **vacation** | — | vacaciones ✓ | OK |
| **enjoy** | disfrutar | disfrutar / gozar de | "We enjoy good performance" → "**Disfrutamos de** un buen rendimiento" |
| **rare** | raro (extraño) | poco común, infrecuente / poco hecho (carne) | "A rare event" → "Un evento **infrecuente**" |
| **strange** | — | raro / extraño ✓ | OK |
| **emphatic** | empático | enfático, rotundo | "An emphatic NO" → "Un NO **rotundo**" |
| **empathic / empathetic** | — | empático ✓ | OK |

| **trillion** | trillón | billón (10¹²) | idem |
| **eventual** | eventual (= temporal) | final, último | "Eventual outcome" → "Resultado **final**" |
| **executive** | ejecutivo (siempre persona) | directivo / ejecutivo / poder ejecutivo | OK contexto |

## 4. Reglas mnemotécnicas rápidas

1. **Si la palabra acaba igual o casi igual, sospecha** (eventually, sensible, library, actually, realize…).
2. **Verifica acrónimos legales/normativos**: en cuanto aparezca uno, mirar si tiene equivalente en el idioma destino.
3. **billón / billion**: **siempre** clarificar a `mil millones` o `billion (10⁹)` si hay riesgo de confusión.
4. **support**: en EN suele significar "dar soporte / admitir / apoyar", no "soportar carga". Si se traduce como "soportar", verificar si el original era "withstand / handle".
5. **realize**: 99% de las veces es "darse cuenta", no "realizar".
6. **eventually**: 99% de las veces es "al final / con el tiempo", no "eventualmente".
7. **actually**: 99% de las veces es "de hecho / en realidad", no "actualmente".

## 5. Casos sensibles donde el error tiene consecuencias

Estos errores **deben** ser cazados antes de entregar la traducción:

- **billion / billón** en cifras financieras o de capacidad (storage, datos): puede generar diferencia de 1000×.
- **switchover / failover** en runbooks de DR: implican procesos diferentes, uno planificado y otro automático.
- **schema / scheme** en doc de BBDD: `scheme` puede sugerir "esquema fraudulento" en EN.
- **molest / molestar**: no es un error técnico pero tiene impacto reputacional grave en EN.
- **constipated / constipado** en doc médica: error grave de sentido.
- **sensible / sensitive** en doc de seguridad: "sensible data" en EN sugiere "datos sensatos" (sin sentido); lo correcto es "sensitive data".
- **compromise** en seguridad: "the system was compromised" = "el sistema fue **vulnerado**", no "el sistema se comprometió".
